## 📋 The 3 Commands

### Take Over
```
/takeover 353871234567
```
Bot pauses, you chat with customer

### Release
```
/release 353871234567
```
Bot resumes, back to automation

### Check Status
```
/status 353871234567
```
See if takeover is active

---

## 📱 Where to Type?

**In WhatsApp!**

1. Open bot's WhatsApp
2. Go to **your own chat** (bot talking to itself)
3. Type command
4. Send!

---

## 🎬 Quick Example

```
1. See customer needs help
   ↓
2. Send to yourself: /takeover 353871234567
   ↓
3. Chat with customer normally
   ↓
4. Send to yourself: /release 353871234567
   ↓
5. Done!
```

---

## ⚠️ Remember

- ✅ Use digits only: `353871234567`
- ❌ No + or spaces: `+353 87 123 4567`
- 📱 Send commands to **yourself**
- 💬 Chat with customer in **their chat**
