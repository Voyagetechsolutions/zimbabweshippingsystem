# 🚀 START HERE

## Quick Test (2 Minutes)

### Step 1: Start the Bot

```powershell
cd ireland-whatsapp-bot
npm start
```

### Step 2: Scan QR Code

A QR code will appear in the terminal. 

**On your phone:**
1. Open WhatsApp
2. Go to **Settings** → **Linked Devices**
3. Tap **"Link a Device"**
4. Scan the QR code from your computer screen

### Step 3: Test the Bot

**IMPORTANT:** You need to test from a DIFFERENT phone!

**From another phone**, send a WhatsApp message:
```
hi
```

To the number you just connected (your test number).

The bot should respond with:
```
🇮🇪 Zimbabwe Shipping Ireland 🇿🇼

Welcome! How can I help you today?

Main Menu:
1️⃣ 📦 Book a Shipment
2️⃣ 💰 View Pricing
3️⃣ 🔍 Track Shipment
4️⃣ 📍 Collection Areas
5️⃣ ❓ FAQ & Help
6️⃣ 📞 Contact Us

Reply with a number (1-6) to continue.
```

## ✅ Success!

If you see the menu, the bot is working! 🎉

## 🧪 Test Commands

Try these from the other phone:

- `hi` - Show main menu
- `2` - View pricing
- `4` - Collection areas
- `6` - Contact info

## 🚨 Troubleshooting

### "I don't have another phone to test"

Options:
1. Ask a friend to send you a message
2. Use WhatsApp Web on another browser
3. Use a second SIM card

### "Bot not responding"

1. Check terminal - should say "BOT CONNECTED SUCCESSFULLY"
2. Make sure you're testing from a DIFFERENT phone
3. Check for error messages in terminal

### "QR code expired"

1. Stop bot (Ctrl+C)
2. Run `npm start` again
3. Scan new QR code quickly
